Add Bangaroo to Your Browser

Google Chrome

  1. Right-click on the address bar and select Edit search engines.
  2. Under "Other search engines," click Add.
  3. Fill in the following:
    • Search engine: Bangaroo
    • Keyword: bangaroo (or any shortcut you prefer)
    • URL: Paste the URL copied from above
  4. Click Add to save.
  5. To make it your default search engine, find Bangaroo in the list, click the three dots, and select "Make default".

Mozilla Firefox

  1. Right-click on the search bar at the top of Firefox.
  2. Select Add a Keyword for this Search.
  3. Fill in the following:
    • Name: Bangaroo
    • Keyword: bangaroo (or any shortcut you prefer)
    • URL: Paste the URL copied from above
  4. Click Save.
  5. To set as default, go to Settings > Search > Default Search Engine and select Bangaroo.

Microsoft Edge

  1. Go to Settings (click the ... menu in the top right).
  2. Select Privacy, search, and services from the sidebar.
  3. Scroll down to Address bar and search and click on it.
  4. Click Add search engine.
  5. Fill in the following:
    • Search engine: Bangaroo
    • Keyword: bangaroo
    • URL with %s in place of query: Paste the URL copied from above
  6. Click Add.
  7. To make it your default, find Bangaroo in the search engine list and click "Make default".

Safari

Unfortunately, Safari doesn't allow custom search engines.

Brave

  1. Go to Settings > Search engines.
  2. Scroll to the bottom and click Add.
  3. Fill in the following:
    • Search engine: Bangaroo
    • Keyword: bangaroo
    • URL with %s in place of query: Paste the URL copied from above
  4. Click Add.
  5. To make it your default, find Bangaroo in the list and click the three dots, then "Make default".
